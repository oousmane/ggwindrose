# Hourly wind observations

Hourly wind speed and direction records

## Usage

``` r
wind
```

## Format

A data frame with 6 variables:

- year:

  Integer. Year of observation.

- month:

  Integer. Month of observation (1–12).

- day:

  Integer. Day of observation (1–31).

- hour:

  Integer. Hour of observation (0–23, UTC).

- wspd:

  Numeric. Wind speed (m/s).

- wdir:

  Numeric. Wind direction (degrees, 0 = North, clockwise).

## Source

ERA5 hourly reanalysis sample data.
